# Orval Setup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all hand-written API functions, React Query hooks, and TypeScript interfaces with Orval-generated code using React Query + Zod validation.

**Architecture:** Single Orval config generates React Query hooks (with Zod validation) into `src/api/generated/` and Zod schemas + TS types into `src/models/`. A custom mutator adapter delegates all HTTP calls to the existing `apiClient` Axios instance.

**Tech Stack:** Orval, React Query (TanStack v5), Zod, Axios, Next.js 15

### Task 1: Install Orval and create config

**Files:**

- Modify: `package.json`
- Create: `orval.config.ts`

- [ ] **Step 1: Install orval as a dev dependency**

Run: `yarn add -D orval`

- [ ] **Step 2: Create `orval.config.ts`**

```ts
import { defineConfig } from 'orval'

export default defineConfig({
  compliance: {
    input: {
      target: 'http://100.81.155.97:8000/api/schema/',
    },
    output: {
      target: 'src/api/generated/',
      client: 'react-query',
      mode: 'tags-split',
      schemas: 'src/models/',
      prettier: true,
      override: {
        mutator: {
          path: 'src/api/mutator/custom-instance.ts',
          name: 'customInstance',
        },
        zod: {
          generate: {
            response: true,
            body: true,
            param: true,
            query: true,
            header: false,
          },
        },
        header: (info) => [
          `Generated by orval`,
          `Do not edit manually.`,
          ...(info.title ? [info.title] : []),
        ],
      },
    },
    hooks: {
      afterAllFilesWrite: 'prettier --write',
    },
  },
})
```

### Task 2: Create custom mutator

**Files:**

- Create: `src/api/mutator/custom-instance.ts`

- [ ] **Step 1: Create `src/api/mutator/custom-instance.ts`**

```ts
import apiClient from '@/api/api-client'
import type { AxiosError, AxiosRequestConfig } from 'axios'

export const customInstance = <T>(config: AxiosRequestConfig): Promise<T> => {
  const promise = apiClient(config).then(({ data }) => data)
  return promise
}

export default customInstance

export type ErrorType<Error> = AxiosError<Error>
```
